Instructions for creating favicon.ico:

  1. Take screenshots of each icon size below (use browser dev tools for precise capture)
  2. Save each size as a separate PNG file
  3. Use a favicon generator tool (like favicon.io) to convert the PNGs into a single .ico file
  4. Save the resulting file as favicon.ico in the root directory
